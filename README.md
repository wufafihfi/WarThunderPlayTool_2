# 战雷小工具
- 这个程序可以让你快速且精准地测量战雷M键小地图两点间距离
## 测距模式
- 对局内按住`M`然后在游戏弹出的地图上点击`鼠标右键`,点击两处位置即可测量其间距
## 校准模式
- 每个地图的比例尺不一样像素长度也不一样，所以需要校准以适配不同地图
- 1.先在程序目录下找到 `config.json` 并打开
- 2.根据游戏内给出的比例尺实际距离调整 `mapDistance_m` 的数值
- 3.在游戏内按下`M`+`CTRL`然后分别在比例尺两端按下`鼠标右键`连线，线越直越好，尽量点准一点
## 样式调整
- 在启动程序后，屏幕右侧回出现一个蓝色圆边正方形，使用鼠标拖出来
- 关闭程序请按窗口右上角的`×`
- 窗口可收纳至屏幕边缘处任意位置
- 文字字体大小 圆点大小 可以到程序目录下的 `config.json` 并找到 `TextSize` 和 `circle_R`
